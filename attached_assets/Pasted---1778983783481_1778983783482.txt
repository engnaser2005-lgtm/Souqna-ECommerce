قم الآن بإنشاء نظام المنتجات الكامل والاحترافي للبائعين.

========================
إضافة المنتجات
==============

أنشئ صفحة Add Product احترافية للبائع فقط.

الحقول المطلوبة:

* اسم المنتج
* وصف المنتج
* السعر
* القسم
* صور متعددة
* الكمية المتوفرة
* حالة المنتج
* المدينة
* رقم التواصل
* هل المنتج مميز Featured
* نسبة الخصم إن وجدت

========================
رفع الصور
=========

المطلوب:

* رفع صور متعددة
* Preview للصور قبل الرفع
* حذف الصور قبل النشر
* ضغط الصور وتحسين الأداء
* رفع الصور إلى Supabase Storage

استخدم Bucket:
product-images

========================
صفحة تفاصيل المنتج
==================

أنشئ صفحة Product Details احترافية تحتوي على:

* معرض صور احترافي
* معلومات المنتج
* السعر
* التقييم
* حالة المنتج
* بيانات البائع
* منتجات مشابهة
* زر شراء
* زر استعلام

========================
صفحة منتجات البائع
==================

أنشئ Seller Dashboard للمنتجات.

البائع يستطيع:

* مشاهدة جميع منتجاته
* تعديل المنتج
* حذف المنتج
* إخفاء المنتج
* معرفة عدد المشاهدات
* معرفة عدد الاستفسارات
* معرفة عدد الطلبات

========================
البحث والفلاتر
==============

أضف:

* بحث مباشر
* فلترة حسب القسم
* فلترة حسب السعر
* فلترة حسب المدينة
* ترتيب حسب:

  * الأحدث
  * الأعلى سعراً
  * الأقل سعراً
  * الأكثر مشاهدة

========================
قاعدة البيانات
==============

تأكد أن جدول products يحتوي على:

* id
* seller_id
* title
* description
* price
* images TEXT[]
* category
* quantity
* city
* contact_number
* condition
* featured
* discount_percentage
* views_count
* inquiries_count
* orders_count
* created_at
* updated_at

========================
تحسينات مهمة
============

أضف:

* Pagination
* Infinite Scroll أو Load More
* Skeleton Loading
* تحسين الأداء
* Lazy Loading للصور
* رسائل نجاح وفشل احترافية

========================
الحماية
=======

* فقط البائع يستطيع تعديل منتجاته
* فقط البائع يستطيع حذف منتجاته
* المشترون لا يستطيعون الوصول لصفحة الإضافة

========================
المطلوب التقني
==============

* استخدام React Query أو أفضل طريقة لإدارة البيانات
* تنظيم الخدمات داخل services
* Hooks مخصصة للمنتجات
* كود نظيف وقابل للتوسع

تأكد أن النظام احترافي بالكامل ويشبه المتاجر العالمية.
